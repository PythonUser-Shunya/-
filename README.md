# 切り抜き
動画内の動いている部分のみを切り取って画像として保存するプログラムです。教授の要望があったので作りました。

研究室活動の一環で森に設置した自動撮影カメラの映像（撮影対象は動物）を用いました。

与えられた動画フォルダの階層を崩さずに画像を保存する部分が難しかったです。
